# Form Validation

A simple login form built while learning form validation with vanilla JavaScript.

## Features

- Username and password validation
- Show/hide password toggle
- Login success/error messages
- Redirects to dashboard on successful login

## Files

- `index.html` - Login form
- `dashboard.html` - Dashboard page (shown after login)
- `script.js` - Form validation logic
- `dashboard.js` - Dashboard authentication check
- `data.js` - Mock user data
- `style.css` - Styling
